Règlement Échec Et Math :


Le Pion peut se déplacer de deux cases dès son premier déplacement et seulemnt d'une case tout le reste de la partie. Le pion ne peut seulement prendre une autre pièce en diagonale. Dans l'image si dessous nous avons représenté les déplacements possible que le pion peut faire.

La Tour se déplace horizontalement ou verticalement. Dans l'image si dessous nous avons représenté les déplacements possible que la tour peut faire.

Le Cavalier se déplace seulement en forme de "L". Dans l'image si dessous nous avons représenté les déplacements possible que le cavalier peut faire.

Le Fou se déplace en diagonale. Dans l'image si dessous nous avons représenté les déplacements possible que le fou peut faire.

La Reine se déplace horizontalement, verticalement ou en diagonale. Dans l'image si dessous nous avons représenté les déplacements possible que la reine peut faire.

Le Roi se déplace d'une case disponible autour de lui. Dans l'image si dessous nous avons représenté les déplacements possible que la roi peut faire.
