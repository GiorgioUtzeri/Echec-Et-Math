Échec Et Math :


Echec et Math est un jeu d'échec où vous pouvez jouer contre un ami présent sur le même écran que vous (nous prévoyons un mode multi mais cela n'est pas sûr).

Le but est donc de créer ce site en seulement 8 mois pour faire cela et nous sommes à fond sur ce projet.

Le code ne connait pas les règles c'est à dire que dans le jeu vous pouvez poser un pion où vous souhaitez. Nous considéront que les joueurs connaissnt le déplacement des pions et les règles du jeu

Pour commencer à jouer rien de plus simple : cliquer sur le logo à droite de ce texte.

Voilà pour les informations concernant le site et ce projet. Pour plus d'info cliquez sur "en savoir plus"

Bon jeux à tous !
